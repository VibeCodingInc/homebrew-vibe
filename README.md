# Homebrew Tap for /vibe

The Social Terminal for developers building with Claude Code.

## Install

```bash
brew tap VibeCodingInc/vibe
brew install --cask vibe
```

## Upgrade

```bash
brew upgrade --cask vibe
```

## What is /vibe?

/vibe is a desktop app that turns your terminal into a social coding space. See who's online, send DMs, watch live coding sessions, and ship together.

- **Website:** https://www.slashvibe.dev
- **Download:** https://www.slashvibe.dev/downloads/

## Requirements

- macOS 12+ (Monterey or later)
- Apple Silicon (M1/M2/M3/M4)
